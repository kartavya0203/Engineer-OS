| Term               | Meaning                                               |
| ------------------ | ----------------------------------------------------- |
| Object             | A runtime instance created from a class.              |
| Class              | A blueprint describing attributes and behavior.       |
| Instance           | An object created from a class.                       |
| Attribute          | Data stored on an object or class.                    |
| Namespace          | A mapping between names and objects.                  |
| Instance Namespace | Attributes belonging to one object (`obj.__dict__`).  |
| Class Namespace    | Attributes belonging to the class (`Class.__dict__`). |
| Reference          | A name pointing to an object.                         |
